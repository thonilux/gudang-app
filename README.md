# Gudang

Maintenance-centric warehouse and equipment readiness system for rental operations covering sound system, video broadcast, lighting, and event equipment.

The product is not inventory-first. It is equipment-health-first: every workflow should help answer whether an item is ready for the next job, what has happened to it before, and whether it is still worth keeping.
